**RedGrid** is a full-stack web application designed to bridge the gap between blood donors, hospitals, and camp organizers. Built on the MERN stack, it serves as a digital infrastructure to manage blood inventories, organize donation camps, and facilitate emergency inter-hospital transfers.

## üöÄ Key Features

### üë§ For Users (Donors & Recipients)
- **Live Inventory Search:** Check availability of specific blood groups in nearby hospitals.
- **Smart Requests:** Raise emergency blood requests with document verification.
- **Gamified Rewards:** Earn **Bronze, Silver, and Platinum** badges based on donation frequency.
  - *Perk:* Unlock free health checkups at partner hospitals based on tier.
- **Camp Alerts:** Receive geo-location based notifications for nearby donation drives.

### üè• For Hospitals (Blood Banks)
- **Inventory Management:** Real-time updates of blood stock.
- **Inter-Hospital Network:** Request stock from other registered hospitals during critical shortages.
- **Document Verification:** Admin panel to verify donor request documents.

### ‚õ∫ For Organizations
- **Camp Management:** Organize and list donation camps with location coordinates.
- **Donor Outreach:** Automated notifications to users within a specific radius of the camp.
